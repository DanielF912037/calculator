## Simple GUI Calculator
This is a basic graphical user interface (GUI) calculator built in Python using the Tkinter library. The calculator can perform addition, subtraction, multiplication, and division operations.

## Table of Contents

- Features
- Requirements
- How use
- License

## Features

Addition, subtraction, multiplication, and division operations.

A user-friendly interface.

Ability to handle division by zero.

## Requirements

Python 3.x

Tkinter (usually included with Python)

## How use]

Run the script calculator.py using Python:

Enter two numbers in the input fields.

Select the desired operation from the dropdown menu.

Click the "Calculate" button to perform the calculation.

The result will be displayed on the GUI.

To exit the calculator, simply close the GUI window.
